# JPA-Hibernate-Fundamentals

Comprehensive hands-on practice project covering JPA & Hibernate core concepts.

---

## 📌 Overview

This repository contains practical examples explaining:

- Entity Mapping
- Primary Keys & Generation Strategies
- Relationships (OneToOne, OneToMany, ManyToOne, ManyToMany)
- Entity Inheritance Strategies
- JPQL Queries
- Group By & Aggregation
- Fetch Join & N+1 Problem
- DTO Projection
- EntityManager & Persistence Context

This project is built for deep understanding of how Hibernate works internally.

---

## 🏗 Technologies Used

- Java

- H2 / MySQL (configurable)
